## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/bingwatch.git
cd bingwatch
```

2. **Install dependencies:**

```bash
npm install
# or
yarn install
```

3. **Start the development server:**

```bash
npm run dev
# or
yarn dev
```

4. **Open** [http://localhost:3000](http://localhost:3000) in your browser.
